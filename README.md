# SmolEngine

![Alt Text](https://i.imgur.com/fy8LmB6.gif)

SmolEngine is a 2D game engine that is developed primarily by one person for a learning purposes.


Short-Term Goals:

- 2D Batch Renderer
- ECS
- Editor 
- UI System
- Scripting in C++, Jinx & Lua
- 2D Animations
- 2D Physics
- Particle System
- Serialization
- Post-processing
- Networking
- Audio
