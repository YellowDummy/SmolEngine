SmolEngine is an open-source 2D game engine that is developed primarily by one person for learning purposes.

Short-Term Goals:

- 2D Batch Renderer (OpenGL / Vulkan)
- Entity component system (ECS)
- Editor 
- UI System
- Scripting in C++, Jinx & Lua
- Animations (2D)
- Physics (2D)
- Particle System
- Serialization
- Post-processing
- Networking
- Audio
