# SmolEngine


![alt text](https://i.imgur.com/JGy78b9.png "SmolEngine")

SmolEngine is a 2D game engine that is developed primarily by one person for a learning purpose.


Short-Term Goals:

- 2D Batch Renderer
- ECS
- Editor 
- UI System
- Scripting in C++, Jinx & Lua
- 2D Animations
- 2D Physics
- Particle System
- Serialization
- Post-processing
- Networking
- Audio
